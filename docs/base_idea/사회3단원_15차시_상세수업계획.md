# 배움의 덩어리 1 — 한정된 자원, 현명한 선택 [4사07-01]

## 1차시 · 창업 지원 사업 출범식 + 우리 도시 알아가기 (준비하기)

**〈획득 개념〉** 경제활동 — *생활에 필요한 물건을 만들고 사용하는 것과 관련된 모든 활동* / (우리 도시의 특성·강점 = 뒤의 사업 아이템과 교류·상호의존의 복선)

**〈이 차시의 배움〉** 단원 전체가 하나의 '창업 프로젝트'임을 이해하고 자신을 **우리 도시의 창업가**로 받아들인다. 이어서 **"이 도시에서 사업을 하려면 우리 도시부터 알아야 한다"**는 동기로 우리 도시의 특산품·자연환경·강점을 조사해 안다.

| 단계 | 활동 |
|---|---|
| 도입 | 교사가 '시장' 위촉장을 들고 등장 → "오늘부터 나는 OO시 시장입니다. 우리 도시를 키울 기업을 찾습니다." 도시 깃발·간판으로 교실을 도시로 변신. |
| 전개 | ① 교과서 단원 도입(장터 그림)으로 경제활동의 예 찾기 → '만들고 사고파는 것이 경제활동'임을 정리<br>② **우리 도시 선정**(회장단이 뽑은 5개 도시 중) → 도시 이름·위치 확인<br>③ **우리 도시 알아가기** : "이 도시에서 사업하려면 우리 도시부터 알자" → 특산품·자연환경·강점 조사(개인 → 판서 → 교사 보충으로 목록 확정) |
| 정리 | "이 강점으로 무엇을 만들면 좋을까?"를 마음에 품기 → 다음 차시 예고 |

**탐구 질문** [사실] 우리 도시는 어떤 특성과 강점을 가진 곳일까?
**과정중심평가** 핵심어 찾기·도시 조사 관찰, 역할 수용 정도 점검
**준비물** 위촉장·도시 깃발/간판, 단원 도입 PPT, 도시 조사 자료(태블릿/지역 누리집)

> 변경 메모 : 특산품 조사를 2차시에서 **1차시로 이동**. "선택이 필요해서 특산품을 조사한다"는 연결은 논리적 비약이라, '사업 무대인 우리 도시를 알아본다'는 자연스러운 동기 아래로 옮겼습니다. 덕분에 2차시는 희소성 한 줄기로 또렷해집니다.

## 2차시 · 다 만들 수 있을까? — 희소성 체감 (탐구하기)

**〈획득 개념〉** 자원의 희소성 — *원하는 것은 많지만 돈·시간 등 자원이 부족한 상태* → 그래서 **선택의 문제**가 생긴다.

**〈이 차시의 배움〉** 우리 회사가 *하고 싶은 것*과 *실제 가진 자원*을 맞부딪쳐 보며, **자원이 한정돼 다 할 수 없다(희소성)**는 것을 자기 사업 상황에서 체감한다. 그래서 **선택이 필요하고, 잘 고르려면 기준이 있어야 함**을 깨닫는다.

| 단계 | 활동 |
|---|---|
| 도입 | 1차시 우리 도시 강점을 떠올리며 "이 강점으로 만들고 싶은 것 다 적어보기"(브레인스토밍, 많이!) |
| 전개 | ① **하고 싶은 것 vs 가진 것 충돌** : 만들고 싶은 목록 옆에 제약 얹기 — "예산 10만 원, 제작 기간 며칠, 일손도 한정". 다 할 수 있나? → **못 한다 = 희소성** 정리<br>② **개념 일반화** : 우리만 그런가? 가게·도시·나라도 마찬가지 → 교과서 사례(용돈으로 필통? 카드?)로 확인<br>③ **그래서 선택** : 이 많은 것 중 무엇을 남기고 무엇을 포기할까? → "고르려면 *기준*이 필요하겠다"까지 (방법은 3·4차시) |
| 정리 | "다음 시간엔 *어떻게 잘 고를지*를 배웁니다" → 합리적 선택 예고. (희망 시 사업계획서 1차 초안을 과제·자투리 시간에 시작) |

**탐구 질문** [개념] 우리는 하고 싶은 것을 왜 다 할 수 없을까?
**과정중심평가** '하고 싶은 것–제약' 활동지, 발문 응답 관찰
**준비물** 하고 싶은 것–제약 활동지, 교과서

## 3차시 · 사업체 선정과 회사 만들기 (탐구하기)

**〈획득 개념〉** 합리적 선택의 의미와 중요성 — *자원의 낭비를 막고 더 큰 만족을 얻기 위해, 기준을 세워 선택하는 것*

**〈이 차시의 배움〉** 우리 도시 강점을 살려 **사업계획서를 완성**한 뒤, '시장의 투자 심사'와 '회사의 채용'이라는 두 선택 장면을 통해 **좋은 선택에는 기준이 필요하다**는 것을 이해한다. 합리적 선택이 왜 중요한지를 '한정된 지원금을 잘 써야 한다'는 절박함으로 체득한다.

| 단계 | 활동 |
|---|---|
| 도입 | "어제 깨달은 것 — 다 못 하니 골라야 한다. 그럼 우리 회사는 무엇 하나에 집중할까?" |
| 전개 | ① **사업계획서 완성** : 우리 도시 강점을 살린 만들 것·타겟·차별성·필요 재료 확정. *"특산품을 살리면 지원금 보너스!"* 안내<br>② **사업체 선정** : 반별 4개 회사 선정(특산품 연계 기업 +지원금 1~2만원). 선정 기준을 학생과 함께 세우기 = 합리적 선택 모델링<br>③ **지원금 배분**(회사당 10만원, 모형돈)<br>④ **직원 채용·공무원 임명** : 직원 지원서 제출→면접, 시청 공무원(장부·교류 중개) 임명 |
| 정리 | "이제 돈이 생겼어. 그런데 다 쓰면 안 돼. 무엇부터 사야 할까?" |

**탐구 질문** [개념] 무엇 하나에 집중하고, 한정된 지원금을 어떤 기준으로 나눠야 할까?
**과정중심평가** 사업계획서 피드백(강점 연계·실현 가능성), 선정·채용 기준 발표 관찰
**준비물** 사업계획서 양식, 직원 지원서, 모형돈(천원 단위), 공무원 임명장

> 변경 메모 : 사업계획서 작성을 3차시 앞부분에 **안착**(2차시에서 이동). 2차시의 "선택이 필요하다 → 무엇 하나에 집중"이 곧 사업 아이템 결정으로 이어져, 작성 동기가 또렷해집니다. 분량이 빠듯하면 2차시 끝~과제로 초안을 시작해 3차시 초에 마무리하면 됩니다.

## 4차시 · 무엇부터 살까 — 물품 구입 품의 (정리하기 · ★징검다리 평가 1)

**〈획득 개념〉** 합리적 선택의 방법 — *필요한 것 고르기 → 선택 기준 정하기(가격·품질·필요성) → 정보 수집 → 비교 → 결정*. 그리고 **기회비용**(하나를 고르면 다른 하나를 포기함).

**〈이 차시의 배움〉** 한정된 예산 안에서 **선택 기준을 세워 근거를 들어** 물품 구입 우선순위를 정한다. 포기한 물품과 그 이유를 적으며, 모든 선택에는 대가가 따른다는 것을 이해한다. → **[4사07-01]의 도달 지점**

| 단계 | 활동 |
|---|---|
| 도입 | 시설 이용비 안내(코팅기·프린터 등도 돈) → "10만원으로 다 못 사!" |
| 전개 | ① 아이스크림몰에서 재료·물품 가격 검색<br>② **모둠 토론** : 필요성·가격 기준으로 살 것/뺄 것 결정 → **품의서** 작성(품목·수량·단가·구입처·꼭 필요한 이유)<br>③ '사고 싶었지만 뺀 물품 + 이유' 칸 작성 = 기회비용<br>④ 시장(교사) 결재, 첫 지출액 제출, 장부 기록 |
| 정리 | "왜 그 물건을 빼기로 했어?"를 서로 설명하며 합리적 선택 언어로 정리 |

**탐구 질문** [개념] 한정된 예산을 어디에 먼저 써야 할까?
**★징검다리 평가 1** 품의서 + 토론 / 준거 : ①희소성 인식 ②선택 기준 수립 ③근거 있는 우선순위
**준비물** 품의서 양식, 아이스크림몰 접속 기기, 시설 이용비 안내문, 예산 장부

# 배움의 덩어리 2 — 우리 도시의 생산 [4사07-02]

## 5차시 · 우리 활동은 생산일까 소비일까 (탐구하기)

**〈획득 개념〉** 생산(*물건을 만들거나 서비스를 제공*)과 소비(*물건을 사거나 서비스를 이용*)

**〈이 차시의 배움〉** 자신의 기업 활동을 생산/소비로 **구분**한다. "재료를 산 건 소비, 우리가 물건을 만드는 건 생산"임을 깨달아, 추상적 용어를 자기 경험과 연결한다.

| 단계 | 활동 |
|---|---|
| 도입 | 지난 차시 '재료 구입' 장면 회상 → "그건 생산이었을까 소비였을까?" |
| 전개 | ① 생산·소비 개념 형성(교과서)<br>② **우리 회사 활동 분류표** : 재료 사기·제품 만들기·홍보하기 등을 생산/소비로 나누기<br>③ 생산을 다시 3가지로(자연에서 얻기 / 만들기 / 편리·즐겁게 하기) → 우리 회사는 어디에 속하나 |
| 정리 | "내일부터 우리는 *생산자*가 됩니다!" |

**탐구 질문** [개념] 우리 기업의 활동은 생산일까 소비일까?
**과정중심평가** 분류표 점검, 발문 응답 관찰
**준비물** 생산·소비 분류표, 교과서

## 6차시 · 제작 활동 1 — 생산자가 되다 (실천하기)

**〈획득 개념〉** 생산 활동의 실제 — *노동을 통해 가치가 만들어진다*

**〈이 차시의 배움〉** 맡은 역할대로 제품을 생산하며, **자신의 노동이 곧 생산 활동**임을 몸으로 안다. 성실히 일한 날 일급이 쌓이는 경험으로 노동–소득의 관계를 이해한다.

| 단계 | 활동 |
|---|---|
| 도입 | 역할 확인(제작·홍보 등), 공무원은 시설 예약·지원 준비 |
| 전개 | ① **제품 생산** : 회사별 분업 생산<br>② 공무원 역할 : 어려운 회사 돕기, 시설 관리(필요 시 임시 고용)<br>③ **급여 기록부** 작성 : 성실히 일한 날 일급, 서비스업은 기술 숙련도 기록 |
| 정리 | 오늘의 생산량·어려웠던 점 공유 |

**탐구 질문** [사실] 나는 오늘 어떤 생산 활동을 했나?
**과정중심평가** 급여 기록부, 참여·협업 관찰
**준비물** 구입한 재료, 급여 기록부, 시설(코팅기 등)

**〈획득 개념〉** 생산·소비 구분의 적용 / 책임 있는 노동

**준비물** 재료, 급여 기록부

## 8차시 · 제작 활동 3 — 1차 생산 마무리 (실천하기)

**〈획득 개념〉** 자원·기술의 한계 인식 (→ 교류의 동기)

**〈이 차시의 배움〉** 1차 제품을 완성하면서 **"우리 도시(회사)만으로는 채울 수 없는 것"**이 있음을 절감한다. 이 결핍이 다음 덩어리(교류)의 출발점이 된다.

| 단계 | 활동 |
|---|---|
| 도입 | 완성 목표 확인 |
| 전개 | ① 1차 제품 완성<br>② **결핍 점검표** : 우리에게 부족한 재료·기술·일손은? / 반대로 우리가 잘하는 것은? |
| 정리 | "옆 도시엔 우리에게 없는 게 있다던데?" → 박람회 예고 |

# 배움의 덩어리 3 — 도시와 도시를 잇다 [4사07-02]

## 9차시 · 왜 도시들은 교류할까 + 공무원 도시 소개 모임 (탐구하기)

**〈획득 개념〉** 지역 간 경제 교류(*경제적 이익을 위해 상품·기술·정보를 주고받음*)와 그 **까닭**(*지역마다 자연환경·자원·기술이 다르기 때문*)

**〈이 차시의 배움〉** 우리에게 없는 것을 다른 도시는 가지고 있고, 그 차이 때문에 교류가 일어남을 이해한다. 공무원들이 5개 도시를 처음 한자리에서 비교하며 "박람회를 열자"는 아이디어를 스스로 끌어낸다.

| 단계 | 활동 |
|---|---|
| 도입 | 결핍 점검표 회상 → "왜 도시마다 가진 게 다를까?" |
| 전개 | ① 교류 개념·까닭 형성(교과서·영상)<br>② **공무원 도시 소개 모임** : 5개 반 공무원이 한자리에 모여 각자 도시 카드로 도시 소개 → "우리가 가진 것 / 필요한 것" 공유 → 자연스럽게 "그럼 기업끼리 만나서 교류하면 어떨까?" 아이디어 유도<br>③ 공무원이 반으로 돌아와 박람회 계획 공유 → **기업 소개판** 제작 시작(우리 기업이 잘하는 것·줄 수 있는 것 중심)<br>④ 직원들은 **개인 기술 카드** 작성 : "나는 이걸 잘해요 / 교류 가능한 것" |
| 정리 | "다음 시간엔 다섯 도시 기업이 한자리에 모입니다!" |

**탐구 질문** [개념] 우리 도시에 없는 것은 무엇이고, 다른 도시는 왜 그것을 가지고 있을까?
**과정중심평가** 개인 기술 카드·기업 소개판 점검, 짧은 쓰기
**준비물** 도시 카드(1차시 완성본), 기업 소개판 재료, 개인 기술 카드 양식, 교류 영상

> 자료 구조 메모 (추후 별도 제작)
> - **도시 카드** : 1차시 개인 탐구→반 대표 확정. 공무원이 9차시 모임에 지참. 박람회 부스에도 게시.
> - **기업 소개판** : CEO·직원이 9차시에 제작. 박람회 부스 홍보물.
> - **개인 기술 카드** : 직원 1인 1장. 박람회에서 교류 신청의 실제 대상("저 사람의 저 기술이 필요해").

## 10차시 · 기업박람회 = 교류의 장 (실천하기)

**〈획득 개념〉** 교류의 다양한 모습 — *상품 교류·기술 교류·일손 교류* 등

**〈이 차시의 배움〉** 다른 도시 기업의 소개판과 개인 기술 카드를 보며 **"저 사람의 저 기술이 필요해"** 수준으로 구체적인 교류를 체결한다. 교류의 단위가 도시가 아니라 **기업과 개인**임을 경험한다.

| 단계 | 활동 |
|---|---|
| 도입 | 부스 배치(도시 카드 + 기업 소개판 + 개인 기술 카드 전시), 교류 규칙 안내 |
| 전개 | ① 부스 탐방 : 다른 도시 기업 소개판·개인 기술 카드를 보며 "우리에게 필요한 것" 탐색<br>② **교류 신청서** 작성·체결 : 기업↔기업, 개인↔개인 모두 가능(상품·기술·일손)<br>③ 도움을 받은 기업은 도와준 기업에 **감사 카드** 전달 |
| 정리 | 오늘 성사된 교류 수 집계 |

**탐구 질문** [논쟁] 교류는 누구에게 이득일까? (양쪽 모두인지 토론)
**과정중심평가** 교류 신청서, 교류 태도 관찰
**준비물** 도시 카드·기업 소개판·개인 기술 카드(9차시 완성본), 교류 신청서, 감사 카드

## 11차시 · 교류 정리와 우수기업 시상 (정리하기 · ★징검다리 평가 3)

**〈획득 개념〉** 상호 의존 관계 — *각 지역(기업)은 서로 필요한 것을 주고받으며 의존한다*. 교류의 좋은 점(부족함 보완·발전).

**〈이 차시의 배움〉** 도시들이 **서로 돕는 관계**임을 자기 교류 사례로 설명한다. '많이 판 기업'이 아니라 '**많이 도운 기업**'이 인정받는 경험으로 협력의 가치를 내면화한다. → **[4사07-02] 상호의존의 도달 지점**

| 단계 | 활동 |
|---|---|
| 도입 | "어제 우리는 누구에게 무엇을 받았고, 누구에게 무엇을 줬지?" |
| 전개 | ① **교류 관계도** 그리기 : 화살표로 도시 간 주고받음 표시 → '한쪽이 아니라 서로 의존' 발견<br>② **우수기업 시상** : 감사 카드를 가장 많이 받은 기업에 시장이 **우수기업 인증장** 수여<br>③ "혼자였다면 어땠을까?" 토의 |
| 정리 | 상호의존을 한 문장으로 정리 |

**탐구 질문** [개념] 도시들은 서로 어떤 관계일까?
**★징검다리 평가 3** 교류 신청서·관계도 + 구술 / 준거 : ①교류의 까닭 설명 ②상호의존 사례 설명 ③협력적 참여
**준비물** 교류 관계도 활동지, 우수기업 인증장

# 도착점 — 함께 자란 우리 [전체 종합·전이]

## 15차시 · 장터 개장 · 수익 정산 · 성찰 (정리하기 · ★전이 평가)

**〈획득 개념〉** 단원 전체 종합 — *희소성 → 합리적 선택 → 생산·소비 → 교류 → 상호의존*이 하나의 이야기로 연결됨 = **핵심 문장 도달**

**〈이 차시의 배움〉** 직접 사고팔며 생산자·소비자를 모두 경험하고, 수익을 정산한 뒤 **"우리는 어떻게 선택하고 함께 자랐는가"**를 성찰한다. 배운 개념을 새로운 상황(판매·정산)에 적용·통합한다. → **전이의 완성**

| 단계 | 활동 |
|---|---|
| 도입 | 시장의 개장 선언 |
| 전개 | ① **장터 판매**(A/B팀 교대) — 팔고, 또 손님으로 합리적 구매(영수증·구매 기준 기록)<br>② **수익 정산** : 번 돈 + 남은 예산 = 총 수익 계산<br>③ **성찰 보고** : 가장 잘한 선택 / 교류로 받은 도움 / 아쉬운 점 |
| 정리 | 시장의 질문 "우리는 어떻게 여기까지 왔죠?" → 학생이 핵심 문장으로 답하기. 남은 모형돈은 학기말 아나바다에서 가산 사용 안내 |

**탐구 질문** [핵심] 한정된 자원 속에서 우리는 어떻게 선택하고 함께 발전했나?
**★전이 평가** 수익 정산서 + 성찰 보고 + 자기/동료평가 / 준거 : ①희소성·합리적 선택 근거 ②생산·소비 이해 ③교류·상호의존 설명 ④협력·성찰
**준비물** 모형 돈, 영수증·구매 기록지, 수익 정산서, 성찰 보고서, 교과서 139쪽
