# 📗 경기 상생 창업 프로젝트 — 학생용 가이드

> 이 책자를 읽으면 앱에서 무엇을 해야 하는지 알 수 있어요!  
> 모르는 게 생기면 선생님(시장님)께 물어보세요. 😊

---

## 📱 앱 로그인 방법

1. 크롬북에서 앱 주소를 입력해요
2. 우리 반 도시를 선택해요 (예: 수원시, 이천시 …)
3. 내 번호(출석번호)를 선택해요
4. PIN 4자리를 입력해요

> 💡 **처음 로그인할 때**는 PIN을 직접 정해요. 잊어버리지 않게 기억해 두세요!  
> PIN을 3번 틀리면 30초 동안 잠겨요. 천천히 입력하세요.

---

## 🗺️ 우리 수업은 이렇게 진행돼요

```
⓪ 도시 탐구 → ① 창업 → ② 생산 → ③ 교류 → ④ 판매
```

선생님(시장님)이 단계를 열어줄 때마다 새로운 활동이 시작돼요.  
**잠겨 있는 단계는 아직 우리 차례가 아닌 거예요. 기다려요!**

---

## ⓪ 0단계 — 도시 탐구

**모든 학생이 함께 해요**

우리 도시가 어떤 곳인지 조사하는 단계예요.

### 해야 할 것

| 순서 | 활동 | 메뉴 |
|------|------|------|
| 1 | 경기도 지도에서 우리 도시 찾아 클릭하기 | 홈 → 도시 탐구 |
| 2 | 우리 도시의 특산품 조사해서 입력하기 | 도시 탐구 → 조사 |
| 3 | 우리 도시의 자원 입력하기 | 도시 탐구 → 조사 |
| 4 | 우리 도시의 강점 입력하기 | 도시 탐구 → 조사 |
| 5 | 우리 도시 한 줄 소개 쓰기 | 도시 탐구 → 조사 |
| 6 | **저장하기** 버튼 클릭 | |

> ✏️ **저장 버튼을 꼭 눌러야 선생님께 보내져요!**

---

## ① 1단계 — 창업

이 단계에서 **직업이 정해져요!**  
처음에는 모두 "지원자"로 시작해요.

---

### 🙋 지원자 — 모든 학생이 처음에 해요

**사업계획서를 내면 → 선생님이 선정 → CEO가 됩니다**  
**취업지원서를 내면 → CEO가 채용 → 직원이 됩니다**

#### 사업계획서 작성 (CEO가 되고 싶을 때)

| 순서 | 활동 | 메뉴 |
|------|------|------|
| 1 | 회사 이름 정하기 | 홈 → 창업 → 사업계획서 |
| 2 | 우리 도시 특산품을 활용한 상품 쓰기 | |
| 3 | 판매 대상 쓰기 (예: 같은 반 친구들) | |
| 4 | 우리 도시에 필요한 이유 쓰기 | |
| 5 | **제출하기** 버튼 클릭 | |

> 📌 선생님이 사업계획서를 검토하고 선정해요. 선정되면 **대표이사(CEO)** 가 됩니다!

#### 취업지원서 작성 (직원이 되고 싶을 때)

| 순서 | 활동 | 메뉴 |
|------|------|------|
| 1 | 지원하고 싶은 회사 선택하기 | 홈 → 창업 → 취업 지원 |
| 2 | 자기소개 / 지원 이유 쓰기 | |
| 3 | **지원하기** 버튼 클릭 | |

> 📌 CEO가 취업지원서를 보고 채용 여부를 결정해요.

---

### 👑 대표이사(CEO) — 창업 단계부터

선생님께 사업계획서가 선정된 학생이에요.  
회사를 운영하는 가장 중요한 역할이에요!

#### 직원 채용하기

| 순서 | 활동 | 메뉴 |
|------|------|------|
| 1 | 취업지원서 목록 확인 | 홈 → 창업 → 채용 |
| 2 | 지원자 정보 확인 | |
| 3 | **채용하기** 또는 **다음에** 클릭 | |

> 💡 채용된 학생은 즉시 **직원** 역할로 바뀌고, 화면에 알림 팝업이 떠요!

---

## 직업별 전체 활동 안내

> 이제 직업이 정해졌어요! 내 직업에 해당하는 부분만 읽어보세요.

---

## 👑 대표이사 (CEO)

**회사를 이끄는 역할이에요. 할 일이 가장 많아요!**

| 단계 | 해야 할 일 | 메뉴 |
|------|-----------|------|
| ① 창업 | 직원 채용하기 | 홈 → 창업 → 채용 |
| ② 생산 | 업무일지 작성 (매 수업일) | 홈 → 업무일지 |
| ② 생산 | 품의서 작성 (재료 구입 신청) | 홈 → 품의서 |
| ③ 교류 | 교류카드 등록 | 홈 → 교류 |
| ③ 교류 | 업무일지 작성 | 홈 → 업무일지 |
| ④ 판매 | 상품 등록하기 | 홈 → 회사 관리 |
| ④ 판매 | 판매대 QR 코드 보여주기 | 홈 → 회사 관리 |
| ④ 판매 | 업무일지 작성 | 홈 → 업무일지 |

---

### 📋 품의서 작성하는 법 (재료 구입 신청)

**품의서 = 선생님께 "이것 사도 될까요?" 하고 묻는 문서예요**

1. 홈 → **품의서** 클릭
2. 살 물건 이름 입력 (아이스크림몰 상품명과 똑같이!)
3. **용도** 입력 (예: "쿠키 포장에 쓸 거예요")
4. 수량, 개당 금액 입력
5. 포기한 것 (기회비용) 입력 — 예산이 부족해서 못 산 것과 이유
6. **시장님께 품의서 제출** 클릭
7. 선생님이 승인하면 → 회사 잔액에서 자동 차감!

> ⚠️ **잔액보다 많은 금액은 신청할 수 없어요.** 잔액을 먼저 확인하세요!

---

### 🤝 교류카드 등록하는 법

**교류 = 다른 도시 회사와 서로 협력하는 것**

1. 홈 → **교류** 클릭
2. 우리 회사가 **줄 수 있는 것** 입력 (예: 도자기 그릇 납품)
3. 우리 회사가 **받고 싶은 것** 입력 (예: 쌀 재료 지원)
4. **교류카드 등록** 클릭
5. 공무원이 다른 반을 방문해서 교류를 성사시켜줘요!

---

### 📱 판매 단계 — 상품 등록 & QR 판매

**상품 등록하기:**
1. 홈 → **회사 관리** 클릭
2. 상품 이름, 재고 수량, 가격 입력
3. **추가** 클릭 (최대 8개까지)

**판매대 QR 코드:**
1. 홈 → **회사 관리** → 아래로 스크롤
2. **판매대 QR 코드** 화면이 나와요
3. 이 화면을 친구들에게 보여주면 → 친구가 스캔 → 상품 선택 → 결제!

---

### 📊 자산 흐름 보기

우리 회사 돈이 어떻게 들어오고 나갔는지 볼 수 있어요.

1. 홈 → **회사 관리** → **자산 흐름 보기** 클릭
2. 수입(초록) / 지출(빨강) 타임라인으로 확인

---

## 🛠️ 직원 (Staff)

**CEO와 함께 물건을 만들고, 판매를 돕는 역할이에요.**

| 단계 | 해야 할 일 | 메뉴 |
|------|-----------|------|
| ② 생산 | 업무일지 작성 (매 수업일) | 홈 → 업무일지 |
| ③ 교류 | 업무일지 작성 | 홈 → 업무일지 |
| ④ 판매 | 업무일지 작성 | 홈 → 업무일지 |
| 언제나 | 성찰일지 작성 | 홈 → 성찰일지 |

> 💸 **업무일지를 써야 급여(일당)를 받을 수 있어요!** 매 수업일 꼭 작성하세요.

---

### 📝 업무일지 작성하는 법

**업무일지 = 오늘 내가 한 일을 기록하는 일기예요**

1. 홈 → **업무일지** 클릭
2. 오늘 한 일을 자세히 써요
   - 예: "쿠키 반죽을 만들었어요. 포장지를 접었어요."
3. **저장하기** 클릭

> ✅ 업무일지를 쓰면 선생님 화면에 초록색으로 표시돼요!

---

### 💭 성찰일지 작성하는 법

**성찰일지 = 오늘 배운 것, 느낀 것을 정리하는 일기예요**

1. 홈 → **성찰일지** 클릭
2. 오늘 배운 점, 어려웠던 점, 다음에 더 잘할 것을 써요
3. **저장하기** 클릭

---

### 🛒 물건 사는 법 (구매자 역할)

판매 단계에서 다른 회사 물건을 살 수 있어요!

1. 홈 → **카드** 클릭 (내 잔액 확인)
2. **물건 사기** 버튼 클릭
3. 사고 싶은 회사의 **QR 코드를 카메라로 스캔**
4. 상품 목록에서 원하는 것 선택, 수량 설정
5. **결제하기** 클릭
6. 내 **PIN 4자리** 입력 → 결제 완료!

> ⚠️ 내 회사 물건은 살 수 없어요. 다른 회사 QR만 스캔하세요!

---

## 📋 공무원 (Officer)

**도시의 살림꾼! 기업들이 잘 운영되는지 살피는 역할이에요.**  
선생님(시장님)이 임명해요.

| 단계 | 해야 할 일 | 메뉴 |
|------|-----------|------|
| ② 생산 | 업무일지 작성 | 홈 → 업무일지 |
| ② 생산 | 기업 시찰 보고서 작성 | 홈 → 시찰 보고서 |
| ③ 교류 | 다른 반 방문 → 교류 성사 기록 | 홈 → 교류 |
| ③ 교류 | 업무일지 작성 | 홈 → 업무일지 |
| ④ 판매 | 업무일지 작성 | 홈 → 업무일지 |
| 언제나 | 성찰일지 작성 | 홈 → 성찰일지 |

> 💰 공무원도 매 수업일 일당을 받아요. 업무일지를 꼭 쓰세요!

---

### 🔍 기업 시찰 보고서 작성하는 법

**시찰 = 회사를 직접 방문해서 잘 운영되고 있는지 확인하는 것**

1. 홈 → **시찰 보고서** 클릭
2. 방문한 회사 선택
3. 회사에서 본 것을 기록해요
   - 어떤 물건을 만들고 있나요?
   - 회사 분위기는 어떤가요?
   - 칭찬할 점, 개선할 점은?
4. **보고서 제출** 클릭 → 선생님(시장님)께 바로 전달돼요!

---

### 🤝 교류 성사 기록하는 법

다른 반을 직접 방문해서 우리 반 회사와 연결해줘요!

1. 홈 → **교류** 클릭
2. 교류가 이루어진 회사 정보 입력
3. 어떤 교류가 성사됐는지 기록
4. **저장하기** 클릭

---

## 💳 내 잔액 & 카드 보는 법

1. 홈 → **카드** 클릭
2. 내 이름, 잔액, QR 코드가 나와요
3. QR 코드는 **판매자가 스캔**할 때 보여줘요

> 💡 잔액은 급여(일당)를 받거나 물건을 팔면 늘어나고,  
> 물건을 사거나 회사 재료를 구입하면 줄어요.

## ❓ 자주 묻는 질문

**Q. PIN을 잊어버렸어요!**  
A. 선생님께 말씀드리면 초기화해 드려요. 그러면 새로 설정할 수 있어요.

**Q. 업무일지를 못 썼어요. 오늘 급여를 못 받나요?**  
A. 선생님이 급여 지급 버튼을 누르기 전에 쓰면 받을 수 있어요. 빨리 써요!

**Q. 품의서를 냈는데 아무 반응이 없어요.**  
A. 선생님이 결재 중이에요. 잠깐 기다리면 승인/반려 결과가 와요.

**Q. 교류카드를 등록했는데 아무것도 안 해요.**  
A. 공무원이 다른 반을 방문해서 연결해줘요. 공무원 친구를 기다려요!

**Q. QR을 스캔하려는데 카메라가 안 열려요.**  
A. 크롬북 카메라 권한을 허용해야 해요. 주소창 옆 자물쇠 아이콘을 눌러서 카메라를 허용해 주세요.

**Q. 내 직업이 언제 정해지나요?**  
A. 사업계획서가 선정되면 CEO, CEO가 채용하면 직원, 선생님이 임명하면 공무원이 돼요. 직업이 정해지면 화면에 팝업이 나타나요!
