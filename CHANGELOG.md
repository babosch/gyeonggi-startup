## 2026-06-19

### UX
- **학생 홈 개편** — 현재 단계 작업을 도시 컬러 큰 카드로 부각, 이전 단계는 작게, 미래(잠긴) 작업은 숨김. "지금 할 일"이 한눈에.

### 운영·관리
- **슈퍼어드민** 추가 (`/admin/super`) — 반별 시장 현황·해제, 잘못 등록된 교사 계정 정리.
  `SUPER_ADMIN_EMAIL` 환경변수로 지정된 교사만 접근.
- **시장 등록 친절한 안내** — 이미 시장이 있는 반에 등록 시 명확한 한글 메시지(중복키 에러 대신).
- 루트(`/`) 페이지가 기본 Next 템플릿이던 버그 → `/home` 리다이렉트로 수정.

### 협업 체계
- **CI 추가** (`.github/workflows/ci.yml`) — PR·push 시 자동 타입체크+빌드. "의미 충돌"(삭제된 함수 호출 등) 검출.
- **협업 가이드** (`docs/협업가이드.md`) — 브랜치 흐름, 충돌·합치기 순서, DB 동기화, main=실서비스 규칙.

### 배포·성능
- **Vercel 배포** (gyeonggi-startup.vercel.app), 함수 리전 서울(`icn1`) 고정으로 속도 개선.
- **keep-alive** GitHub Actions — 3일마다 Supabase ping(7일 정지 방지). 동작 확인 완료.
- 학생 홈 카드를 "오늘 할 일 / 언제든지 해요" 두 섹션으로 정리.

### AI 연속성 문서
- `docs/ARCHITECTURE.md`(코드 구조·패턴·함정), `docs/DECISIONS.md`(결정 배경) 추가.
